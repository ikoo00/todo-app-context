##  Deskripsi Aplikasi

Todo App Context adalah aplikasi manajemen tugas (todo list) yang dibangun menggunakan React Native. Aplikasi ini memungkinkan pengguna untuk membuat, mengelola, dan menyaring daftar tugas mereka dengan fitur prioritas dan pencarian. Aplikasi ini menggunakan React Context API untuk state management yang efisien.

---

##  Fitur Utama

### 1. **Tambah Todo**
   - Input teks untuk menulis todo baru
   - Tombol tambah cepat dengan icon "+"
   - Support untuk todo tanpa batas

### 2. **Prioritas Todo**
   - Tiga level prioritas: High (Merah), Medium (Biru), Low (Hijau)
   - Pemilihan prioritas saat menambah todo
   - Visual indicator untuk prioritas setiap item

### 3. **Filter Todo**
   - **Semua (All)** - Menampilkan semua todo
   - **Aktif (Active)** - Menampilkan todo yang belum selesai
   - **Selesai (Done)** - Menampilkan todo yang sudah selesai

### 4. **Manajemen Todo**
   -  Tandai todo sebagai selesai/belum selesai
   -  Hapus todo individual
   -  Statistik: Jumlah todo selesai vs total

### 5. **Hapus Bulk**
   - Tombol "Hapus [n] item selesai" untuk menghapus semua todo yang sudah selesai sekaligus

### 6. **Dark Mode**
   - Toggle tema gelap/terang
   - Interface yang nyaman di mata untuk penggunaan malam hari
   - Transisi warna yang smooth
